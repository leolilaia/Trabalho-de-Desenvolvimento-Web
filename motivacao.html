Motivação do Projeto

Este projeto foi criado para mostrar como a trigonometria pode ser usada na prática para calcular a altura de um prédio a partir da distância e do ângulo.

Como foi feito

Usamos HTML para estruturar a página, JavaScript para realizar os cálculos, e um arquivo CSS separado para o visual.

Por que foi feito

O foco é demonstrar a aplicação prática da trigonometria, não apenas suas fórmulas teóricas. O objetivo é usá-la para determinar a altura de estruturas (ex: prédios). Busca-se tornar o conceito acessível, evidenciando sua conexão direta com o mundo real.
